# 📘 Sistema de Cadastro de Usuários em C

Aplicação simples desenvolvida em **C** para simular um sistema de cadastro de usuários via terminal, com foco no aprendizado de lógica de programação e manipulação de dados.

---

## 🚀 Funcionalidades

- Cadastro de usuários  
- Listagem de usuários cadastrados  
- Armazenamento de informações básicas  
- Interação via terminal com menu  
- Manipulação de dados em memória  

---

## 🧠 Conceitos aplicados

- Estruturas (`struct`)  
- Manipulação de arrays  
- Entrada e saída de dados (`scanf` / `printf`)  
- Estruturas condicionais (`if/else`)  
- Laços de repetição (`for`, `while`)  

---

## 🛠️ Tecnologias

- Linguagem C  

---

## ▶️ Como executar

```bash
git clone https://github.com/lucasvieirafelisberto/C-Sistema-de-Cadastro-de-Usuario.git
cd C-Sistema-de-Cadastro-de-Usuario
gcc main.c -o programa
./programa
